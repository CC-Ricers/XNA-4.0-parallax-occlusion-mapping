
- Parallax Occlusion Mapping demo - 

Este es en realidad un ejemplo doble. Por un lado se muestra el efecto
llamado Parallax Occlusion Mapping (usado en el Crysis), que permite dar 
a una superficie plana la sensación de que tiene relieve "real".
Por otro, se ha modificado el content processor de XNA para poder
cargar más de una textura a la vez para cada parte del modelo y así
poder usar este tipo de efectos.



- Controles -

Teclas W/S/A/D: traslada la cámara

Ratón: Rota la cámara

Tecla Espacio: cambia el modo de pintado de texturas. Los modos disponibles
	son:

	· Texture mapping
	· Normal mapping
	· Parallax Occlusion Mapping

Tecla Enter: activa / desactiva el sistema de partículas
